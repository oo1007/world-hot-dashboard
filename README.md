# World Hot Dashboard

Daily global news, investment, and market dashboard generated by Codex.

- `index.html`: latest dashboard
- `world_hot_dashboard_YYYY-MM-DD.html`: daily archive
- `skill/world-hot/`: Codex skill used to collect global news and trend sources

## Install the Skill

From a local clone of this repository on Windows:

```powershell
Copy-Item -Recurse -Force .\skill\world-hot "$env:USERPROFILE\.codex\skills\world-hot"
```

Then call it in Codex as `$world-hot`.
